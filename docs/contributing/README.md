# Contributing to CO2

Thank you for your interest in contributing to CO2! There are many ways to contribute and all of them are appreciated.

## Stay in touch

This doc probably won't cover everything. Feel free to open an issue for asking questions.
You can also join [our telegram group](https://t.me/c_rust_interop) for real time communication.

## Contribution ideas

* Use `co2cc` in your C projects and report bugs
* Convert your C project to a `co2cargo` project, report bugs and missing features
* Run `co2miri` on those projects
* Improve missing docs

## Building from source

For doing code contributions, you need to build CO2 from source, which is a non trivial thing to do since it depends on a custom rustc toolchain.
You can find [the instructions here](../installation.md). That's a bit incomplete, feel free to ask me in case you had problems.
